# Stardew Clock

Stardew Clock is a small client-side Fabric mod for Minecraft that adds a cozy Stardew Valley-inspired clock overlay to your HUD.

The mod brings the familiar warm pixel-art feeling of Stardew Valley’s clock into Minecraft while keeping the information useful for regular gameplay. It displays the current in-game time, your player coordinates, the current time of day, and the current weather in a compact HUD widget.

## Features

- Stardew Valley-inspired pixel-art clock overlay
- Numeric Minecraft time display
- Rotating clock hand that follows the in-game day cycle
- Time-of-day indicator
- Weather indicator for sunny, rainy, and stormy weather
- Player coordinates displayed directly on the clock
- Client-side only
- Lightweight and simple

## Displayed Information

The clock shows:

- Current in-game time
- Player coordinates
- Time of day, such as morning, day, evening, or night
- Current weather, including sunny, rain, or thunderstorm

## Configuration

The mod includes simple configuration options for the HUD overlay. You can enable or disable displayed information and adjust the clock position on the screen.

If Mod Menu is installed, the configuration screen can be opened from the Mods menu.

## Compatibility

Stardew Clock is built for Fabric and is intended to run as a client-side HUD mod. It does not add blocks, items, world generation, or server-side gameplay changes.
